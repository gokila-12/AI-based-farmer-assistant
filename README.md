# AI Based Farmer Assistant Platform

AI Based Farmer Assistant Platform is an intelligent agriculture support system developed using Artificial Intelligence and Machine Learning technologies. The platform helps farmers with crop disease prediction, soil analysis, weather updates, market price information, farming schedules, government schemes, and multilingual farmer support through a user-friendly web application.

---

# Features

## Crop Leaf Disease Prediction

* Detects crop leaf diseases using AI and Machine Learning
* Helps farmers identify diseases quickly and accurately
* Supports better crop management and productivity

## Weather Information and Updates

* Provides real-time weather updates and forecasts
* Helps farmers plan agricultural activities effectively

## Nearby Market Price and Location

* Displays nearby market locations
* Shows crop market prices
* Helps farmers make better selling decisions

## Soil Recommendation System

* Allows farmers to select their soil type
* Provides soil fertility details
* Suggests water requirements for crops
* Recommends suitable crops based on soil conditions
* Supports improved soil and crop management

## Farmer Calendar

* Provides farming schedules and seasonal guidance
* Available for different states across India
* Helps farmers manage cultivation activities efficiently

## Government Schemes Information

* Displays agriculture-related government schemes
* Provides state-wise scheme details for farmers

## Farmer Support Chatbot

* Rule-based chatbot system for farmer assistance
* Answers common farming-related queries
* Provides farming guidance and support

## Multi-Language Support

* Supports Google Translate integration
* Users can select their preferred language on the login page
* The entire interface changes to the selected language
* Chatbot responses are also generated in the selected language
* Improves accessibility for farmers from different regions

# Installation

```bash id="4a5p2h"

```

## 2. Navigate to the Project Directory

```bash id="q4sk4j"

```

## 3. Install Required Dependencies


